D&D Dice Roller

How to Roll

  1. Select the dice you want to roll.
  2. Click the dice as many times as needed (e.g., click twice if you need 2 dice).
  3. Add more dice if necessary.
  4. Press the Run button to roll (Tip: click a dice in the output area to remove it).
